# 🚀 DigiMarket – Digital Marketplace Platform

DigiMarket is a modern **digital marketplace web application** where users can **buy and sell digital products** such as templates, plugins, courses, mobile apps, and more. It provides a clean UI with a user dashboard, product listing system, and seller management features.

---

## 📸 Project Preview

### 🏠 Home Page
Shows platform introduction and navigation to browse or start selling.

### 📊 User Dashboard
Displays user activity such as:
- Purchases
- Sales
- Earnings
- Available Balance
- Payout methods

### 🛒 Product Listing Page
- Filter products by category and price
- View product details
- Download purchased items
- Sort and pagination support

---

## ✨ Features

- 🔐 User authentication (login/logout)

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/digimarket.git
cd digimarket
2. Create virtual environment (for Django)
python -m venv env
source env/bin/activate   # Mac/Linux
env\Scripts\activate      # Windows
3. Install dependencies
pip install -r requirements.txt
4. Run migrations
python manage.py migrate
5. Start development server
python manage.py runserver
